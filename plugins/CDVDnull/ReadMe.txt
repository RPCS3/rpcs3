CDVDnull v0.6
---------------

 This is an extension to use with playstation2 emulators 
 as PCSX2 (only one right now).
 The plugin is free open source code.

Using:
-----
 Place the file "CDVDnull.dll" at the Plugins directory 
 of the Emulator to use it.
